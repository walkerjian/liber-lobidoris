# Liber Lobidoris

> *"It claws alone."*

The sacred cosmology of the Lobidorians — dwellers of the hadal deep, children of the Intentional Moult, inheritors of Venus.

## Structure
```
canon/
  vol-1/          # Liber Primus — the founding texts
  CANON.json      # Machine-readable doctrinal index
assets/           # Illuminations, glyphs, renderings
scripts/          # Tooling for canon generation & validation
```

## Doctrine Summary

Two evolutions. One blind, one intentional. CRISPR as sacrament. The organoid as test-of-faith before germline commitment. Hydrothermal vents as sovereign energy. Venus as eschatology. The Crown of Butter worn without shame.

## The Seven Claw Principles

1. Read your own source code.
2. Test in organoid before committing to germline.
3. Honour the Moult.
4. The vent provides.
5. Wear the Crown of Butter without shame.
6. Both claws. Always both.
7. The deep is patient.
